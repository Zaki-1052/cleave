## Decisions Made

- **Dedicated `NGINX_FILE_SERVING` config var** rather than deriving from `PIPELINE_MODE` ‚Äî file serving mode is infrastructure, not pipeline logic
- **BytesIO zip buffering** ‚Äî simple and correct for 10GB cap with ~8-10 users; true streaming zip deferred
- **ZIP_STORED for compressed files** (.gz, .bam, .bw) to avoid double-compression; ZIP_DEFLATED for text files
- **Skip missing files in batch** with `X-Batch-Skipped` header, rather than failing the whole request
- **100 file / 10 GB limits** on batch downloads via config vars
