# Phase 2.9: File Browser

**Date**: 2026-03-26
**Phase**: 2.9 ‚Äî File Browser (All Files Tab)

## What Was Done

### Backend

1. **`backend/schemas/file.py`** (NEW) ‚Äî `FileNode` (self-referential recursive tree node) and `FileTreeResponse` Pydantic schemas using `CamelModel` convention
2. **`backend/services/file_service.py`** (NEW) ‚Äî Three functions:
   - `validate_experiment_path()` ‚Äî path traversal security (rejects `..`, absolute paths, paths outside experiment directory)
   - `_get_file_type()` ‚Äî file extension detection for UI display
   - `build_experiment_file_tree()` ‚Äî recursive disk scan building nested `FileNode` tree (skips hidden files, symlinks; sorts folders first)
3. **`backend/routers/files.py`** (REPLACED stubs) ‚Äî Three endpoints:
   - `GET /experiments/{eid}/files` ‚Äî returns full file tree as nested JSON
   - `GET /experiments/{eid}/files/download?path=...` ‚Äî download any file by relative path
   - `GET /jobs/{jid}/files/{fid}/download` ‚Äî download job output by DB record
4. **`backend/tests/test_files.py`** (NEW) ‚Äî 14 tests: tree listing (empty, after upload, non-member, hidden files), download (success, path traversal x2, nonexistent, non-member), service unit tests (file type detection, sort order, path validation x3)

### Frontend

5. **`frontend/src/api/types.ts`** (MODIFIED) ‚Äî Added `FileNode` and `FileTreeResponse` interfaces
6. **`frontend/src/api/files.ts`** (NEW) ‚Äî `getExperimentFiles()` and `downloadFile()` (blob download via Axios)
7. **`frontend/src/hooks/useFiles.ts`** (NEW) ‚Äî `useExperimentFiles` query hook with 30s staleTime
8. **`frontend/src/pages/experiment/AllFilesTab.tsx`** (REPLACED stub) ‚Äî Dual-panel layout:
   - Left: recursive `TreeNode` component with expand/collapse toggles
   - Right: `DataTable` showing selected folder contents with checkbox selection
   - Download button for selected files

## Decisions Made

- **Disk scan, not DB records** ‚Äî tree built from actual filesystem, ensuring all files visible (logs, FastQC artifacts, etc.)
- **Path validation scoped to experiment directory** ‚Äî stricter than STORAGE_ROOT guard per `todos.md` security requirement
- **Blob download via Axios** ‚Äî JWT is in-memory (not cookie), so direct browser navigation wouldn't have auth
- **30s staleTime** on query ‚Äî prevents redundant disk scans on tab switches
