## Decisions Made

- Peak caller and peak size are job-level settings (single selection for entire job, not per-reaction) — matches CUTANA Cloud behavior and backend validation
- IgG auto-detection: reactions with "igg" (case-insensitive) in short_name are pre-selected as control
- BAM path resolution: fetches alignment job outputs with `useJobOutputs(alignmentJobId, 'unique_bam')` and matches by `reactionId`
- Reference genome inherited from parent alignment job (read-only in peak calling wizard)
- Advanced settings show only the threshold relevant to the selected peak caller (conditional rendering)
- FRiP color coding: green ≥0.2, amber 0.1-0.2, red <0.1
